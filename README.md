# bootprint
An esoteric programming language with minimal structures

An example program:
```
// initialise
x <= 5                  // set x to 5
y <= 0                  // set y to 0

// looping
false == (x == y) {     // while x is not equal to y
    y <= y + 1          // increment y
}

// io
terminal <= y           // print y to the terminal
```
